# hubot-ama

A hubot script that does the things

See [`src/ama.coffee`](src/ama.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-ama --save`

Then add **hubot-ama** to your `external-scripts.json`:

```json
["hubot-ama"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
